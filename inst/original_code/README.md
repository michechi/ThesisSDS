# Original Reference Code

This directory contains the original implementations preserved for reference.

- `geng_mfm_sbm.R` — Adapted MFM-SBM implementation from Geng et al. (2019)
- `geng_original.R` — Original unmodified code from Geng et al. (2019)

These files are **not** part of the package's exported functionality.
Use the functions in `R/` instead.
